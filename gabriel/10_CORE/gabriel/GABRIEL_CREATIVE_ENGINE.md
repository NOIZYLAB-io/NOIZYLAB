# GABRIEL CREATIVE ENGINE (Tools)

**Mission**: Produce artifacts, not advice.

**Core Tool Suite**:

1. **Audio Audit**: LUFS/TP, RMS, Crest, Harshness, Mono/Stereo, Phase.
2. **Stem Pipeline**: Separate -> Label -> Align -> Export -> Report.
3. **Noise Repair**: De-noise/click/clip + A/B Compare.
4. **Mix Notes Engine**: Detect Masking -> Propose EQ/Comp moves.
5. **Reference Match**: Compare Track vs Ref -> Change List + Deltas.
6. **Video Brain**: Scene Detect, Silence Cut, Highlights, Captions.

**"Creative Intent" Protocol**:

- **Input**: Vibe ("darker, wider, punchier").
- **Output**: Actionable Moves + Comparable Versions.

**Format**:

- **Input**: File Hash + Directive.
- **Output**: Artifact MemCell (Rendered File) + Diff/Report.
