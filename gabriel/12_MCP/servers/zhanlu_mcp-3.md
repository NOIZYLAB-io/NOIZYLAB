# MCP 介绍

> MCP（模型上下文协议）让湛卢与外部数据源和工具集成，使得湛卢可与外部数据源间建立通信，从而扩展湛卢能力。

## 配置MCP

以下是为 playwright 设置单个MCP Server的示例配置

```
{
  "mcpServers": {
    "playwright": {
      "command": "npx",
      "args": [
        "@playwright/mcp@latest"
      ]
    }
  }
}
```

了解MCP Server更多信息，[请参阅文档](https://ecloud.10086.cn/op-help-center/doc/article/78884)。
