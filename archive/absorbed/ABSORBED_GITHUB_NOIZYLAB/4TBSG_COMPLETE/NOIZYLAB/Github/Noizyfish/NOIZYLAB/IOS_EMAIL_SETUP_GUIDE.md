### **Setup Each Account Manually:**

#### **For fishmusicinc.com emails:**

1. **Settings ‚Üí Mail ‚Üí Accounts ‚Üí Add Account**
2. **Select "Other"**
3. **Tap "Add Mail Account"**
4. **Enter:**
   - Name: Your Name
   - Email: rp@fishmusicinc.com or info@fishmusicinc.com
   - Password: Your email password
   - Description: fishmusicinc.com

5. **Choose IMAP**
6. **Enter Server Settings:**
   - **Incoming Mail Server:**
     - Host Name: mail.fishmusicinc.com
     - Username: rp@fishmusicinc.com (or info@fishmusicinc.com)
     - Password: Your email password
   - **Outgoing Mail Server:**
     - Host Name: mail.fishmusicinc.com
     - Username: rp@fishmusicinc.com (or info@fishmusicinc.com)
     - Password: Your email password

7. **Advanced Settings:**
   - Incoming: Port 993, Use SSL
   - Outgoing: Port 587, Use TLS

8. **Repeat for info@fishmusicinc.com**

#### **For noizylab.ca emails:**

Same process, but use:
- Email: rsp@noizylab.ca, help@noizylab.ca, or hello@noizylab.ca
- Host Name: mail.noizylab.ca
- Same ports: 993 (IMAP), 587 (SMTP)

#### **For rsplowman@icloud.com:**

1. **Settings ‚Üí Mail ‚Üí Accounts ‚Üí Add Account**
