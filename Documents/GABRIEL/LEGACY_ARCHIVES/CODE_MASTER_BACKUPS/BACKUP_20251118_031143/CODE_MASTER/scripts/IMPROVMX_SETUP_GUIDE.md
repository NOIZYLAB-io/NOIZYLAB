## Step-by-Step Setup (Voice/Keyboard Accessible)

### Step 1: Sign Up (Free Account)

1. **Open ImprovMX:**
   - Already open at: https://app.improvmx.com/
   - Or say: "Hey Siri, open ImprovMX"

2. **Sign Up:**
   - Use keyboard: TAB to navigate
   - Find "Sign up" or "Create account" button
   - Press SPACE or RETURN to click
   - Fill in form using voice dictation or keyboard:
     - Email: `rsplowman@gmail.com`
     - Password: (create secure password)
   - TAB to "Create account" button
   - Press RETURN

**Voice Commands:**
- Say: "Click Sign up"
- Say: "Type [your email]"
- Say: "Type [your password]"
- Say: "Click Create account"

---

### Step 2: Add Your Domains

1. **Navigate to Domains:**
   - After signup, you'll see dashboard
   - TAB to "Add domain" or "Domains" section
   - Press SPACE to click

2. **Add First Domain:**
   - TAB to domain input field
   - Type: `fishmusicinc.com`
   - TAB to "Add" or "Continue" button
   - Press RETURN

3. **Verify Domain:**
   - ImprovMX will show verification instructions
   - You'll need to add a TXT record to your DNS
   - **BUT** - since you want to bypass DNS changes, ImprovMX has an alternative:
     - Look for "Skip verification" or "Verify later" option
     - Some features work without full verification

4. **Add Second Domain:**
   - Repeat for: `noizylab.ca`

**Voice Commands:**
- Say: "Click Add domain"
- Say: "Type fishmusicinc.com"
- Say: "Click Add"
- Say: "Type noizylab.ca"
- Say: "Click Add"

---

### Step 3: Create Email Aliases (Forwarding)

1. **Navigate to Aliases:**
   - TAB to "Aliases" or "Email forwarding" section
   - Press SPACE to click

2. **Add First Alias:**
   - TAB to "Add alias" or "Create alias" button
   - Press SPACE
   - In "Alias" field, type: `rp`
   - TAB to "Forward to" field
   - Type: `rsplowman@gmail.com`
   - TAB to "Save" or "Create" button
   - Press RETURN

3. **Add Remaining Aliases:**
   - Repeat for:
     - Alias: `rsp` → Forward to: `rsplowman@gmail.com`
     - Alias: `help` → Forward to: `rsplowman@gmail.com`

**Voice Commands:**
- Say: "Click Add alias"
- Say: "Type rp"
- Say: "Type rsplowman@gmail.com"
- Say: "Click Save"

---

### Step 4: Configure SMTP (For Sending - Optional)

1. **Navigate to SMTP Settings:**
   - TAB to "SMTP" or "Send emails" section
   - Press SPACE

2. **Configure SMTP:**
   - ImprovMX provides SMTP settings
   - You can use these in Gmail "Send as" configuration
   - Or just use Gmail's built-in "Send as" (easier)

## Keyboard Navigation Shortcuts

- **TAB:** Move to next field/button
- **SHIFT+TAB:** Move to previous field/button
- **SPACE:** Click button/checkbox
- **RETURN:** Submit form/click primary button
- **ESC:** Cancel/close dialog
- **CMD+F:** Find/search on page
- **Arrow Keys:** Navigate dropdowns/lists

## Troubleshooting

### If Domain Verification Required:
- ImprovMX may ask for DNS verification
- Look for "Skip" or "Verify later" option
- Basic forwarding may work without full verification
- Contact ImprovMX support if needed

### If Can't Find Buttons:
- Use CMD+F to search page
- Type button name (e.g., "Add domain")
- Use Voice Control "Show numbers" command
- Use TAB to cycle through all clickable elements

### If Form Won't Submit:
- Make sure all required fields are filled
- Check for error messages (usually in red)
- Try pressing RETURN instead of clicking
- Use Voice Control "Click Submit" command
