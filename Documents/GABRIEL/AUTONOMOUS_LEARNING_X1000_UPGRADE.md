## ðŸŽ¯ HOW TO USE

### Basic Usage

```python
from autonomous_learning import AutonomousLearning

# Initialize system
al = AutonomousLearning()

# Register a learner
learner = await al.register_learner('john_doe', {
    'name': 'John Doe',
    'learning_style': 'visual',
    'interests': ['ai_ml', 'programming'],
    'career_goals': ['ml_engineer']
})

# Assess knowledge
assessment = await al.assess_knowledge('john_doe', 'ai_ml', {
    'python': 0.9,
    'machine_learning': 0.6,
    'deep_learning': 0.3
})

# Generate personalized curriculum
path = await al.generate_curriculum(
    'john_doe',
    'machine_learning',
    time_available=10  # hours per week
)

# AI Tutor interaction
response = await al.ai_tutor_chat(
    'john_doe',
    'machine_learning',
    'What is gradient descent?'
)

# Record learning session
session = await al.record_learning_session(
    'john_doe',
    'gradient_descent',
    duration_minutes=45,
    comprehension_score=0.85,
    exercises_completed=5
)

# Get analytics
analytics = await al.get_learning_analytics('john_doe')

# Find learning partners
partners = await al.find_learning_partners('john_doe', 'ai_ml')

# Optimize career path
career = await al.optimize_career_path('john_doe', 'ML Engineer', 'Tech')
```

### Advanced Features
