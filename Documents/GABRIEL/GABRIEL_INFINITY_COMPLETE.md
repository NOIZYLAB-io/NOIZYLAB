## ğŸ”§ TROUBLESHOOTING

### **Common Issues & Solutions**

#### **Issue 1: Port Already in Use**
```bash
# Error: Address already in use: 8080
# Solution: Use a different port
python3 gabriel_infinity.py --port 8081

# Or find and kill the process
lsof -ti:8080 | xargs kill -9
```

#### **Issue 2: Import Errors**
```bash
# Error: ModuleNotFoundError: No module named 'X'
# Solution: Install missing dependencies
pip install -r requirements.txt

# Or install specific package
pip install librosa torch numpy
```

#### **Issue 3: Permission Denied**
```bash
# Error: Permission denied when accessing files
# Solution: Fix permissions
chmod +x gabriel_infinity.py
chmod -R 755 /Users/rsp_ms/GABRIEL
```

#### **Issue 4: Memory Error**
```bash
# Error: MemoryError or system slowdown
# Solution: Reduce memory usage in config
# Edit config.json: "memory_size": 5000
# Or close other applications
```

#### **Issue 5: Cloud Connection Failed**
```bash
# Error: Failed to connect to AWS/Azure/GCP
# Solution: Check credentials and network
export AWS_ACCESS_KEY_ID=your_key
export AWS_SECRET_ACCESS_KEY=your_secret
# Test connection: aws s3 ls
```

### **Debug Mode**
```bash
# Run with full debug output
python3 gabriel_infinity.py --debug --verbose

# Check logs
tail -f logs/gabriel.log

# Test individual systems
python3 -m pytest tests/ -v
```

### **Getting Help**
- Check logs in `logs/gabriel.log`
- Run diagnostics: `python3 gabriel_infinity.py --diagnose`
- Review system status: Use `status` command in interactive mode
- Enable verbose logging: Set `GABRIEL_LOG_LEVEL=DEBUG`
