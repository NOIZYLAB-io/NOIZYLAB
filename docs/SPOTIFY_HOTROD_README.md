## ğŸ”¬ Audio Features Explained

### BPM (Tempo)
- **60-90:** Slow, ballads, ambient
- **90-120:** Medium, pop, hip-hop
- **120-140:** Fast, dance, house
- **140+:** Very fast, drum & bass

### Energy (0-100%)
How intense/active the track feels.
- **0-30%:** Calm, relaxing
- **30-60%:** Moderate
- **60-80%:** Energetic
- **80-100%:** Intense, aggressive

### Danceability (0-100%)
How suitable for dancing.
- Based on tempo, rhythm, beat
- **70%+:** Club-ready
- **50-70%:** Moderately danceable
- **0-50%:** More listening-focused

### Valence (0-100%)
Musical positivity/mood.
- **70-100%:** Happy, cheerful
- **40-70%:** Neutral
- **0-40%:** Sad, angry, tense

### Key & Mode
- **Key:** 0-11 (C, C#, D... B)
- **Mode:** Major (happy) or Minor (sad)
- **Harmonic mixing:** Use same or adjacent keys

### Instrumentalness
- **High:** Mostly instrumental
- **Low:** Lots of vocals

### Acousticness
- **High:** Acoustic instruments
- **Low:** Electronic/synthesized
