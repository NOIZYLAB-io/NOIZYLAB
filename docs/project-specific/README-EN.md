# Fitten Code: Your Professional AI Coding Assistant

Fitten Code is an AI coding assistant powered by a large-scale code model developed by Fitten Tech. It supports multiple languages, including Python, JavaScript, TypeScript, Java, C, C++, and more. With Fitten Code, you can automatically complete code, generate code, generate comments, edit code, explain code, generate tests, find errors, and more, all within the sidebar of Visual Studio Code.

---

Fitten Code aims to make your coding experience more enjoyable and efficient.

## Features

### Code Autocompletion

Fitten Code automatically completes your code by predicting what comes next. It can save you valuable development time, increase your work efficiency, accelerate your development process, and provide the convenience of seamless code completion.

- Press tab to accept all completion suggestions.
- Press CTRL+⬇️ (Windows, Linux) / CMD+⬇️ (macOS) to accept a line of completion suggestions.
- Press CTRL+➡️ (Windows, Linux) / CMD+➡️ (macOS) to accept a single word completion suggestion.

![Fitten Code Image](https://code.fittentech.com/assets/images/readme/completion.gif)

### Code Translation

Fitten Code provides semantic-level code translation between multiple languages.

### Comment Generation

Automatically add line-level comments to your code, saving you valuable development time.

![Fitten Code Image](https://code.fittentech.com/assets/images/readme/doc.gif)

### More Features

#### Code Explanation

Select the code, ask questions about it in the chatbox. Fitten Code will explain to you.

#### Test Generation

Fitten Code can generate test cases for the code you select.

#### Error Finding

Fitten Code can find potential defects in your code.
