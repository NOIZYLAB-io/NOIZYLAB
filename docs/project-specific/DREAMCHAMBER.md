## üîë API KEYS SETUP

**You'll need API keys for the providers you want to use:**

### Anthropic (Claude)
- Go to: https://console.anthropic.com/settings/keys
- Create key
- You already have: `sk-ant-api03-jdXjxMTODL...`

### OpenAI (GPT)
- Go to: https://platform.openai.com/api-keys
- Create key
- Format: `sk-...`

### Google (Gemini)
- Go to: https://aistudio.google.com/app/apikey
- Create key
- Format: `AIza...`

### Together AI (Llama)
- Go to: https://api.together.xyz/settings/api-keys
- Create key
- Format: starts with alphanumeric

### Mistral AI
- Go to: https://console.mistral.ai/api-keys
- Create key
- Format: starts with alphanumeric

### Cohere
- Go to: https://dashboard.cohere.com/api-keys
- Create key
- Format: starts with alphanumeric

### Perplexity
- Go to: https://www.perplexity.ai/settings/api
- Create key
- Format: `pplx-...`

**On first use, DREAMCHAMBER will ask for keys.**  
**They're stored in browser localStorage - never sent to Rob's servers.**
