# 01_CORE - Core Gabriel Infrastructure

Core infrastructure files for the Gabriel AI system.

## Contents

This directory contains:
- Core Python modules
- Main Gabriel framework files
- Essential libraries and utilities

## Key Files

Place the following types of files here:
- `GABRIEL_*.py` - Main Gabriel modules
- `gabriel_*.py` - Core functionality
- `cli.py` - Command line interface
- `daily_intel.py` - Daily intelligence processing

## Subdirectories

- `gabriel_main/` - Root Python files
- `gabriel_core_main/` - Core main modules
- `gabriel_memcell/` - Memory cell integration
- `gabriel_root/` - Root utilities
